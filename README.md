## Stack
Pure HTML/CSS/JS — no framework, no build step, no server.  
Progress saved to `localStorage` (persists on GitHub Pages).

## Weekly Review
Every Monday at 9am PDT, a GitHub Action creates a new Issue with a weekly reflection + planning template. Close it when you're done with the review.

## Daily Snapshots
Every day at 6am PDT, a commit is made to `snapshots/` as an audit trail.  
Export your board state from the Quest Board → paste into that day's snapshot file.
